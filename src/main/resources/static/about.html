What is this?

This is a Spring Boot application that serves static HTML content. It demonstrates how to create a modern, responsive web application using Spring Boot's built-in static resource serving capabilities.

Features

  • Modern CSS with gradients and animations
  • Responsive design for all devices
  • Interactive JavaScript functionality
  • Beautiful UI with smooth transitions
